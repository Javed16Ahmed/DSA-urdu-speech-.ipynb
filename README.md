This project is a Google Colab-based Urdu Speech Intent Classification system that uses Machine Learning / Deep Learning techniques to classify Urdu speech or text into predefined intent categories.

It is designed to help in building intelligent Urdu language applications such as chatbots, voice assistants, and NLP-based automation systems.

🚀 Features
Urdu speech/text intent classification
End-to-end ML pipeline in Google Colab
Data preprocessing for Urdu language
Model training and evaluation
Easy-to-run notebook format
Suitable for beginners in NLP and AI
🧠 Technologies Used
Python 🐍
Google Colab
Pandas & NumPy
Scikit-learn / TensorFlow / PyTorch (as used in your notebook)
NLP Techniques
